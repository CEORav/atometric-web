Standard Configurations

Maximum Table Load

5 lbs (2.73 kg);
Higher payload, as special.

Servo Spindles

1000 RPM Servo air cooled spindle as standard,
200,000 RPM water chilled servo spindle, specific higher power or alternate RPM spindles.

Speed and Acceleration


/Decs XYZ of >2 G’s; Speed maximum of 30 m/min.

Resolution

Repeatability

Conductive Contact Probing


(Standard) Contact probing within one micron. G-codes supporting application usage.

Travel


X 101 mm (4 inch) work envelop. Y 101 mm (4 inch) work envelop. Z 76 mm (3 inch) stroke. 101 mm (4 nch) work envelop w/ table rotation.

File Format


G code program storage in .txt files. CAM option programs semi-auto generated from SolidWorks, ACAD, STEP files.

User Interface
